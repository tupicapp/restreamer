# Mindset

- Prioritize maintainable, extendable, modular design so swapping third-party
  services, deploy patterns is easy.
- Keep code as simple as possible.
- Write tests for every module and struct to ensure other parts can use them
  correctly.
- .md files are your mindset and principles that you use to implement anything.
- add any assumption you made in read me
- when i ask you to implement something don't implement if there is an ambiguity and clearify them all before implementing
